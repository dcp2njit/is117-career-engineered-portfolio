### Option 1: Deploy to GitHub Pages (Recommended)

```bash
# 1. Navigate to portfolio folder

# 2. Initialize git

git commit -m "Initial portfolio commit"

# 3. Create repo on GitHub.com called: denisepayumo.github.io
```
